使用前，请先安装CTEX套装最新full版，或者texlive最新版。
参考文献管理使用JabRef,帖子里有链接
a3cover中放的是a3封面
data中放的是各章内容，务必以utf-8格式保存
figures是文档所用插图
ref放的是文章要用的参考文献为.bib格式需要下载JabRef2.4，有需要可以找我要
thesis.tex是主文档，在这里可以填写封面信息并增删章节
chdpaper.pdf为示例文档
nudtpaper-2.2.pdf是文档代码详细说明，因为我是从国防科技大学的模板改过来的，就借用这个了。
所有文档编辑完后，保存，双击makepdf.bat黑框出来，祈祷不会出现错误。运行完毕，会出现thesis.pdf文档，这个就是你的论文
开题报告中注意\myoverlay（画边框）的放置，不能放在开题报告第一页，因为第一页是单独画边框的
部分程序仿真使用code环境，请注意换行，否则文字会超出页边
